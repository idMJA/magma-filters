# magma-filters
A plugin that allows you to set filter faster in Kazagumo

Install
```
npm i magmafilters
```

Support filter:
```
- clear
- eightD
- soft
- speed
- karaoke
- nightcore
- pop
- vaporwave
- bass
- party
- earrape
- equalizer
- electronic
- radio
- tremolo
- treblebass
- vibrato
- china
- chimpunk
- darthvader
- daycore
- doubletime
- pitch
- rate
- slow
```
